# 공무원연금공단 상담 통계 대시보드

`공무원연금공단_연금콜센터 상담내역정보_20221231.csv` 데이터를 기반으로 만든 정적 대시보드입니다.

## 파일 구성
- `index.html`: 대시보드 화면
- `style.css`: UI 스타일
- `script.js`: CSV 로드/집계/렌더링 로직
- `data/pension_callcenter_20221231.csv`: 웹용 UTF-8 데이터 파일

## 기능
- 연도, 상담유형 필터
- 총 상담건수/레코드 수/유형 수/세분류 수 KPI
- 연도별 상담건수
- 상담유형/상담분류 TOP 통계
- 상담세분류 TOP 테이블
